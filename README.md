### 默认用户名 ROOT 默认密码 password 默认lan口ip 192.168.8.1

### 自动云编译步骤

每周一/周五凌晨3点21分自动编译
或fork仓库，点击右上角star，标记为starred后自动开始编译，后续需取消，然后再次点击star开始编译
